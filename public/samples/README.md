# Sample videos (cameraless testing)

Drop any short clip of a face here (e.g. `face.mp4`) and choose **Sample video**
mode in the app to exercise face tracking + ROI extraction without a webcam —
handy on a machine that has no camera.

- Video files (`*.mp4`, `*.webm`, `*.mov`) are gitignored, so they stay local.
- For believable pulse readings the clip should show a reasonably still,
  evenly-lit face.
- The fully **Synthetic** mode needs no video at all and is the primary way to
  verify the signal pipeline end-to-end without hardware.
